C) CSS Files and Structure - top

Styles

I'm using 9 CSS files in this theme. I've separated styles for improving convenience to edit this template. If you're going to use this template in production I'd recommend you merge all stylesheets in one file (for decreasing http queries).

Lets take a look closely at our stylesheet files.

font-awesome.min.css

Base styles for FontAwesome icon fonts.

animate.css

Animate.css main file to perform animation using CSS3.

superslides.css

Styles for Superslides Slider.

bootstrap.min.css

The default Bootstrap 3 styles.

main.css

Common css styles for this template.

colors/green.css

Stylesheet that applies selected color. Please include one of the provided colors.

custom-styles.css

Please put all your custom changes to this file. This CSS has to be loaded last to override perfectly.

F) FAQs - top

How to work with contact form?

This template contains fully working contact form. All you need is replace you actual e-mail address. Open the send-email.php file that is located in the root with any text editor and replace all instances of "your-email@yourdomain.com" with your email address in the entire file using the Find and Replace tool (usually the Ctrl+H hot key).
