C) CSS Files and Structure - top

Styles

I'm using 10 CSS files in this theme. I've separated styles for improving convenience to edit this template. If you're going to use this template in production I'd recommend you merge all stylesheets in one file (for decreasing http queries).

Lets take a look closely at our stylesheet files.

normalize.css

Many browser interpret the default behavior of html elements differently. By using a general reset CSS file, we can work round this. This file also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.

skeleton.css

This stylesheet contains 12-columns responsive grid system based on skeleton.

base.css

It contains elements for template such as accordion, button and pills, forms and so on.

style.css

Common css styles for this template.

superfish.css

Styles for Multilevel dropdown menu.

prettyPhoto.css

It contains styles for prettyphoto plugin (lightbox-like plugin)

font-awesome.min.css

Base styles for FontAwesome icon fonts.

settings.css

Styles for Revolution slider (used on Home Page).

flexslider.css

Styles for Home Page slider (flexslider2).

media-queries.css

Stylesheet that makes template responsive.

    How to setting up Twitter widget?

    There are two steps to setup the Twitter Widget.

    1. Setup an application at Twitter dev center and get keys.
    2. Put your keys into Twitter files.
    Setup Twitter application
    1. Visit https://dev.twitter.com/apps/ and sign in using your Twitter username and password. This doesn't have to be the username or password of the stream you need access to, just a Twitter account you control.
    2. Select 'Create new application' and enter the application details.
      1. The name and description can be anything you like really, but you can't use 'Twitter' in the name.
      2. The website field can be your main website and doesn't have to be the site where your Twitter feed or feeds are located.
      3. Callback URL can be left blank
    3. Enter the CAPTCHA info and click create
    4. On the next details screen, click 'create my access token'. You may need to refresh the page after a few seconds if it doesn't appear automatically.
    5. Make a note of the Consumer key, Consumer secret, Access token and Access token secret as highlighted below.
    Setup Twitter application
    1. Open get-tweets.php file with help of your editor.
    2. Enter your twitter user name for $twitteruser = "";
    3. Enter your consumer key for $consumerkey = "";
    4. Enter your consumer secret key for $consumersecret = "";
    5. Enter your access token for $accesstoken = "";
    6. Enter your access token secret for $accesstokensecret = "";
    7. Open js/jquery.twitter.js file with help of your editor.
    8. Enter your twitter user name for $screenname = ""; (row 6)
    9. Save your changes.

    How to work with contact form?

    This template contains fully working contact form. All you need is replace you actual e-mail address. Open the sendEmail.php file that is located in the root with any text editor and replace all instances of "YOUR_MAIL" with your email address and YOUR_NAME with your name in the entire file using the Find and Replace tool (usually the Ctrl+H hot key).

    If you don't use gmail account you need to open sendEmail.php and delete this code 

    // GMAIL STUFF

$mail->Mailer = "smtp";
$mail->Host = "smtp.gmail.com";
$mail->Port = 587;
$mail->SMTPSecure = "tls"; 

$mail->SMTPAuth = true; // turn on SMTP authentication
$mail->Username = "YOUR_MAIL"; // SMTP username
$mail->Password = "YOUR_PASSWORD"; // SMTP password

    Also if you have gmail account you need to replace YOUR_PASSWORD with your actual password.

    How to add additional field in Contact form?

    If you need to add additional field (phone field for example) to your Contact form you need to make next steps:

    1. Open contacts.html and find form markup (line 213) then add markup for additional input after some existing field.
    2. Then you need to open sendMail.php and define new variable called $phone after existing variables (line 7) 
      	$phone = $_POST['phone'];
    3. Then replace this code on line 34 
      	$mail->Body = $comments;
      to this one 
      	$mail->Body = "Phone ".$phone."\r\n"; 
	$mail->Body .= $comments."\r\n";

      As you can see you just concatenate new field (Phone) with existing Comments.

    4. And the last thing you need to open js/jquery.form.js and define variable by inserting code after 15 line

      	var phone = $('input#phone').val();

      and add this variable in ajax result. You should see next code:

      	data: 'name=' + name + '&email=' + email + '&subject=' + subject + '&phone=' + phone + '&comments=' + comments,
